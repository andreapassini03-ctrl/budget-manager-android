# Budget Manager – Android Application

Android application for advanced personal finance management, designed to track expenses, analyze spending patterns and generate monthly financial reports.

## Overview

Budget Manager is a modular Android application that helps users manage their finances through multiple integrated components such as expense tracking, calendar-based navigation, statistical charts and automated monthly summaries.

The application focuses on providing clear insights into personal spending behaviour and financial trends.

## Main Features

The screenshots: 
https://andreapassini03-ctrl.github.io/budget-manager-android/

### Expense Management

* Add, edit and delete expenses
* Categorize expenses (food, transport, utilities, etc.)
* Track daily and monthly spending
* Persistent storage of transactions



### Calendar Integration

* Navigate expenses by date
* Visual overview of financial activity
* Quick access to daily records

### Data Visualization

* Interactive charts for spending analysis
* Category distribution graphs
* Monthly trend analysis

### Monthly Reports

* Automatic generation of monthly summaries
* Total income vs expenses
* Category-based breakdown

## Technology Stack

* Android APK / App Bundle
* Kotlin
* Android Studio
* Firebase Login and Database

## Future Improvements

* Cloud synchronization
* Budget planning and limits
* Export data (CSV / PDF)
* Notifications for spending limits
